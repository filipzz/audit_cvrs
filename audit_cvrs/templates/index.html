Audit CVRs Data Site

This site is powered by the audit_cvrs application, which is part of the ElectionAudits software, which provides software and documentation in support of the Principles and Best Practices for Post-Election Audits and the other work of ElectionAudits.org, the nation's clearinghouse for election audit information.

The software is open source, still in very active "alpha" development

Use the manage.py parse command from the command line, then log in to the admin account to get started entering your own data.

The site uses the Django framework. See the Django documentation for background on other ways you can use and modify it.
