Audit CVRs Data Site

This site is powered by the audit_cvrs application, which is associated with the ElectionAudits software, which provides software and documentation in support of the Principles and Best Practices for Post-Election Audits and the other work of ElectionAudits.org, the nation's clearinghouse for election audit information.

The software is open source, still in very active "alpha" development

Set the audit up by using Tools for Comparison Risk-Limiting Election Audits to enter the contest parameters and manifest, enter the random seed, and thus select the ballots to be audited. Copy and paste the "Sorted lookup table" into a file and read it in to this tool using the manage.py parse command from the command line.

Next, log in to the admin account you set up, starting with the CVR data to run the audit. Choose CVRs to assign for auditing, set the status to "assigned", retrieve the proper ballot, record the manual interpretation of the ballot, compare the manual interpretation with the machine interpretation, set the status to "Completed", and set the Discrepancy field to "Interpretations match" or to the proper discrepancy value.

The site uses the Django framework. See the Django documentation for background on other ways you can use and modify it.
